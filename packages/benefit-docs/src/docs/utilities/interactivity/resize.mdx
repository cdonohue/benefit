For any of these `resize` utilities to work, you must apply an [overflow](/layout/overflow) class other than the default `visible` behavior. These example all have `overflow-hidden` applied as an example.

<Grid minWidth="250px">
  <InfoBox pattern="resize" caption="Both">
    <Box className="bg-gray-200 p-4 rounded shadow-lg resize overflow-hidden">
      I can be resized in both directions
    </Box>
  </InfoBox>
  <InfoBox pattern="resize-none" caption="None">
    <Box className="bg-gray-200 p-4 rounded shadow-lg resize-none overflow-hidden">
      I cannot be resized
    </Box>
  </InfoBox>
  <InfoBox pattern="resize-x" caption="Horizontal">
    <Box className="bg-gray-200 p-4 rounded shadow-lg resize-x overflow-hidden">
      I can be resized horizontally
    </Box>
  </InfoBox>
  <InfoBox pattern="resize-y" caption="Vertical">
    <Box className="bg-gray-200 p-4 rounded shadow-lg resize-y overflow-hidden">
      I can be resized vertically
    </Box>
  </InfoBox>
</Grid>
